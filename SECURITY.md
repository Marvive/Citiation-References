# Security Policy

## Supported Versions

We actively maintain the latest release of Logos References.

| Version | Supported          |
| ------- | ------------------ |
| v1.1.x  | ✅ Yes             |
| < v1.1  | ❌ No              |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **do not open a public issue**. Instead, please report it via the following method:

1. Send an email to [marvive42@gmail.com](mailto:marvive42@gmail.com).
2. Include "SECURITY VULNERABILITY: Logos References" in the subject line.
3. Provide a detailed description of the vulnerability and steps to reproduce it.

We will acknowledge your report within 48 hours and work with you to resolve the issue as quickly as possible.
