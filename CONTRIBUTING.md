# Contributing to Logos References

First off, thank you for considering contributing to **Logos References**! It's people like you that make this tool better for everyone in the Bible study community.

## How Can I Contribute?

### Reporting Bugs
- Use our [Bug Report Form](https://github.com/Marvive/Logos-References/issues/new?template=bug_report.yml).
- Describe the bug clearly and provide reproduction steps.

### Suggesting Enhancements
- Use our [Feature Request Form](https://github.com/Marvive/Logos-References/issues/new?template=feature_request.yml).
- Explain why the feature would be useful and how it should work.

### Pull Requests
1. **Fork the repository** and create your branch from `main`.
2. **Setup the environment**:
   ```bash
   npm install
   ```
3. **Make your changes**. If you're adding code, please add tests in `src/__tests__`.
4. **Ensure tests pass**:
   ```bash
   npm test
   ```
5. **Build the project**:
   ```bash
   npm run build
   ```
6. **Submit a pull request** with a clear description of your changes.

## Development Guidelines
- Follow the existing code style (TypeScript, modular structure).
- Ensure all functions are documented with JSDoc where appropriate.
- Keep the UI clean and premium.
